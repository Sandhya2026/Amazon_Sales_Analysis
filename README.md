# 📊 Amazon Sales Analysis

## 📌 Project Description

This mini project analyzes Amazon sales data using Python to extract meaningful business insights. The project focuses on data analysis, visualization, and business recommendations using real-world sales data.

## 🚀 Features

* Data Cleaning
* KPI Analysis
* Sales Analysis by State and Category
* Customer & Product Analysis
* Payment Method Analysis
* Monthly Sales Trend Analysis
* Business Insights & Recommendations
* Data Visualization using Matplotlib

## 🛠️ Technologies Used

* Python
* Pandas
* Matplotlib
* Google Colab
* Microsoft Excel
* GitHub

## 📂 Dataset

* Amazon Sales Dataset
* Total Records: *100,000*
* Features: Orders, Customers, Products, Categories, Payment Methods, Cities, States, and Total Sales.

## 📊 Visualizations

* Sales by State (Bar Chart)
* Category-wise Sales (Pie Chart)
* Monthly Sales Trend (Line Chart)
* Top 10 Products by Sales (Horizontal Bar Chart)

## 📁 Project Structure

* Excelpy.ipynb – Google Colab notebook
* amazonn.xlsx – Dataset
* README.md – Project documentation
